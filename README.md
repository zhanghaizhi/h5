## H5-基础模板

针对移动端添加一些特性：

- 300ms点击延迟
- 模拟滚动
- 上拉加载，下拉刷新
- 路由自动生成
- AppShell
- fontawesome字体图标
- Vuepress项目说明文档
- Eslnit + prettier代码风格控制
- css reset
- 1px细线
- px 转 rem

更多细节一步项目内联的Vuepress文档

### 启动VuePress

内联VuePress托管项目说明文档，运行 VuePress 。

```
yarn install
yarn docs:dev
```

### 项目启动
推荐使用yarn，yarn生成的node_modules结构更为合理。

```
yarn install
yarn serve
```
